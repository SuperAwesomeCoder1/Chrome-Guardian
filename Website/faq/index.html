FAQs

Here you can read the answers to the questions most commonly asked about our Chrome extension.


What is Chrome Guardian?

Chrome Guardian is a Chrome safeguarding extension, which restricts access to certain malicious or otherwise questionable websites. It comes with built-in lists of questonable sites, but additionally allows for the manual blacklisting of websites.

For Whom is Chrome Guardian?

Chrome Guardian is all-inclusive and intended for use by anyone who wants to avoid certain websites. It is especially intended for parents who would like to restrict the websites that their child/children can visit.

How Much Does Chrome Guardian Cost?

Nothing! The beauty of Chrome Guardian is that it is very easy-to-use and versatile, but is also completely free.

How Do I Use Chrome Guardian?

Check out our tutorial page for a brief explanation on how to install and use our chrome extension.

Which Platforms are Supported?

Chrome Guardian supports all platforms which can run the Google Chrome web browser.
